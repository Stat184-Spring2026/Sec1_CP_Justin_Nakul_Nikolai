# Project Plan

## Learning Objectives

The learning objectives of this project are:

1. Develop our coding skills in R, our ability to work in groups, and to get comfortable with presenting to an audience.
2. Effectively work and master the usage of GitHub.
3. Take raw data and turn it into an informative plot/graph.

## Project Goals

The goal for this project is to analyze how crime patterns in Hollywood have changed between 2020 (COVID-19) and 2024 (Post COVID-19). In order to achieve this goal, we divided into
three subgoals: 

1. Explore and compare victim age demographics between 2020 and 2024.

2. Examine and compare different types of crimes between 2020 and 2024.

3. Analyze and compare monthly crime trends in Hollywood between 2020 and 2024.


### Needs

To complete this project, We are going to need effective communication, teamwork, organization, and accountability as a team.
We will also need a credible and publicly available data set regarding the crime incidients in Hollywood. 
Next, we will need to set up a repository in GitHub by cloning the Course Project Template. 
We will then need to create several branches for each teammate that can be used for managing pull requests into the main. 
For data analysis, we will need to R packages such as {ggplot2} package to create a plot/graph, {knitr} and {kableextra} packages to generate and style our tables, and {tidyverse} package to wrangle and tidy the data set.
When submitting our project, we will use GitHub as well as GitHub Desktop to manage the repository, branches, pull requests, commits, and issues.
We will also need to upload the CSV file that is the raw data set, as well as the QMD file and PDF for the reproducible data analysis report. 
We will need to create issues to track our tasks for this assignment. We will need to update the README file using the provided template. 
Finally, at least one graph in png format should be uploaded into the images folder to be used as an interesting insight in the README.


### Steps

1. We’ll mainly work on the project in class and if we need to work on it outside of class we’ll do so during the agreed upon times we stated in the contract (mostly during Tuesday, Thursday, and the weekend before the finals week).
2. We’ll need to create a new repository on GitHub by cloning the given Course Project template in the Stat184-Spring2026 Repo and name it in a Sec#_CP_Name1_Name2_Name3 format. 
3. We’ll need to download the data set regarding the crime that happened in the city of Los Angeles, CA between 2020 and 2024 provided from Data.Gov which was provided by LAPD.
4. We’ll need to wrangle and tidy the data set focusing specifically Hollywood area according to the Tidy Data Principles in RStudio.
5. Since we have three subgoals, we'll need to task one sub goal to each member and each will create a clear and informative data tables and graphs for visualizations.
6. We'll need to refine and improve our codes according to the PCIP method.
7. If every codes are finalized, we'll need to create our own analysis and narration based on the results.
8. We'll need to orepare and deliver the Work-in-Progress Presentation during class time and discuss to our classmates about our insights, challenges, data using pptx.
9. We’ll need to create a reproducible data analysis report using a Quarto file discuss about our data and findings.
10. We’ll need to proofread and finalize our data analysis report and export it into a PDF file.
11. We’ll need to import all our project materials including raw data set, QMD file, png images, PDF file, and R codes etc. into our project GitHub repo.
12. We'll need to update README.md and include relevant visualization from the images folder.
